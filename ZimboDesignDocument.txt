this is a "Zimbo" food-ordering-app kinda similar to Swiggy.
our App Design contains the following components:        

Components Design:
AppLayout
1) Header
    - Logo
    - Navitems

2) Body
    - Search bar
    - RestaurantsContainer
        - RestaurantCard
            -Img
            -Name of Res, Rating, Cuisine, delivery time

3) Footer 
    - copyright
    - links
    - address
    - contact



Features -
Filtering Cards by ratings and Re-rendering using Hooks(useState)


Technologies used: Reactjs, parcel
Testing libraries: Jest, @testing-library/react, @testing-library/dom.

extra dependencies from jest: npm install --save-dev babel-jest @babel/core @babel/preset-env
configure babel.config.js file of jest: 
module.exports = {presets: [['@babel/preset-env', {targets: {node: 'current'}}]]};
